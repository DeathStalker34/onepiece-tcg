# Fase 0 · Setup Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Montar el monorepo del simulador OPTCG con Next.js, Prisma, Vitest, ESLint, CI y pre-commit, dejándolo listo para que la Fase 1 (card data) arranque sin fricción.

**Architecture:** pnpm workspaces + turborepo. Tres workspaces iniciales: `apps/web` (Next.js 14), `packages/engine` (TS puro + Vitest), `packages/card-data` (Prisma schema SQLite). ESLint flat config y Prettier en la raíz. Pre-commit via husky + lint-staged. GitHub Actions corre lint/typecheck/test en cada push.

**Tech Stack:** pnpm 9, Node 20, TypeScript 5.4+, Next.js 14, React 18, Tailwind 3.4, shadcn/ui, Prisma 5 (SQLite), Vitest 1.6, ESLint 9 (flat config), typescript-eslint 8, Prettier 3, husky 9, lint-staged 15, turborepo 2, GitHub Actions.

**Referencias:**

- Spec top-level: `docs/superpowers/specs/2026-04-17-optcg-sim-design.md`
- Calidad innegociable (spec §10): TS strict, engine sin deps de framework, acciones serializables, commits convencionales, 1 PR por fase.

import { cva, type VariantProps } from 'class-variance-authority';

import { cn } from '@/lib/utils';

const buttonVariants = cva(
  'inline-flex items-center justify-center whitespace-nowrap rounded-md text-sm font-medium ring-offset-background transition-colors focus-visible:outline-none focus-visible:ring-2 focus-visible:ring-ring focus-visible:ring-offset-2 disabled:pointer-events-none disabled:opacity-50',

        outline: 'border border-input bg-background hover:bg-accent hover:text-accent-foreground',
        secondary: 'bg-secondary text-secondary-foreground hover:bg-secondary/80',
        ghost: 'hover:bg-accent hover:text-accent-foreground',
        link: 'text-primary underline-offset-4 hover:underline',
      },
      size: {
        default: 'h-10 px-4 py-2',
        sm: 'h-9 rounded-md px-3',
        lg: 'h-11 rounded-md px-8',
        icon: 'h-10 w-10',
      },
    },
    defaultVariants: {
      variant: 'default',
      size: 'default',
    },
  },
);

export interface ButtonProps
  extends React.ButtonHTMLAttributes<HTMLButtonElement>, VariantProps<typeof buttonVariants> {
  asChild?: boolean;
}

const Button = React.forwardRef<HTMLButtonElement, ButtonProps>(
  ({ className, variant, size, asChild = false, ...props }, ref) => {
    const Comp = asChild ? Slot : 'button';
    return (
      <Comp className={cn(buttonVariants({ variant, size, className }))} ref={ref} {...props} />
    );
  },
);
Button.displayName = 'Button';
